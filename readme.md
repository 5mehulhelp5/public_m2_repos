# README

By Kashif Bhatti.
Some of my Magento 2 sample code.

#### There are 5 extensions
1. AliDropShipping - web scraping from AliExpress for dropshipping products
2. ReferaFriend - Enter your and your friend's email address to receive a voucher
3. DreamSearch - Ajax searching products using VueJS [test here](https://magento.preview1.co.uk)
4. Blog - Can add blog posts in admin and add comments from front-end 
5. HomeCarousel - Add images hero carousel slider on the home page from admin 
6. NewProducts - choose categories in admin to show new products with a category tab switcher on the home page 
7. CustomerPhotos - Allow customers to upload their photos from front-end and add a homepage slider of customer photos
8. HomeFeatured - Can edit the homepage featured banners from admin
